# Data Mining & Predictive Analysis (2021)

This project explores both unsupervised and supervised machine learning techniques. It involves segmenting bank customers based on credit card usage and building predictive models to classify individuals based on salary levels.

## üìä Key Problems Solved
1. **Bank Customer Segmentation (Clustering):**
   - Performed EDA and data preprocessing on credit card usage behaviors.
   - Applied **Hierarchical Clustering** and **K-Means Clustering** to identify distinct customer segments for targeted promotional offers.
   - Evaluated clusters using Silhouette scores and Elbow plots.
2. **Salary Classification (Supervised Learning):**
   - Built predictive models to determine if an individual earns more than $50k based on demographic data.
   - Implemented and compared three different algorithms:
     - **CART (Decision Tree)**
     - **Random Forest**
     - **Artificial Neural Networks (ANN)**
   - Conducted performance benchmarking using AUC-ROC curves, Sensitivity, and Precision.
